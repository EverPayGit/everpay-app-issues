# everpay-app-issues
EverPay-MetaChain Pull Issues

If you have any questions, feel free to post them to github.com/EverPayGit/everpay-app-issues.

1. Relay your issue within the repository
2. Maintainers will review your issue and will possibly ask for a in-depth description of said found issue.

*IMPORTANT* Please pay attention to the maintainer's feedback, since its a necessary step to keep up with the standards everpay.network attains to.
Finally, if you're looking to collaborate and want to find easy tasks to start, look at the issues we marked as "Good first issue".
